Menampilkan daftar permintaan layanan

Sesaat setelah login, anda akan berada pada halaman pertama, yaitu List Request. Anda juga dapat memilih icon ( ) untuk mengarahkan ke halaman List Request. Halaman ini berisi card yang merupakan data permintaan layanan yang telah diajukan.

Anda dapat menekan tombol Refresh untuk memperbarui data setelah beberapa waktu.
