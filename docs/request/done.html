Mengubah status permintaan layanan menjadi DONE

Ketika suatu permintaan sudah sukses dilayani dan bantuan sudah dikirimkan ke orang yang membutuhkan. Status layanan diubah menjadi "DONE" dengan tombol Done pada card yang sesuai.

Tunggu hingga proses selesai. Status pada card akan berubah menjadi "DONE".

Anda tidak perlu langsung menekan tombol Refresh untuk memperbarui data, sebab data sudah diperbarui pada server dan tampilan pada aplikasi juga merupakan hasil perubahan terbaru yang anda lakukan.
