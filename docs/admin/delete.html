Admin

Menghapus admin dari daftar

Perlu diketahui bahwa hanya admin dengan role SuperUser yang memiliki akses untuk menambah atau menghapus admin. Admin lainnya hanya memiliki akses untuk melihat list admin yang ada. Pada Datagrid ditampilkan username dan password yang sudah terenkripsi.

Pilih salah satu admin pada DataGrid yang akan dihapus. Username dan password akan tertampil pada input field. Password juga akan diberikan mask.

Anda dapat menekan tombol Cancel untuk membatalkan operasi. Tekan tombol Delete untuk menghapus admin. Tunggu hingga proses selesai.
