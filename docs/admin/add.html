Admin

Menambahkan admin baru

Pada side navigation bar pilih icon () untuk mengarahkan ke halaman Manage Admin. Perlu diketahui bahwa hanya admin dengan role SuperUser yang memiliki akses untuk menambah atau menghapus admin. Admin lainnya hanya memiliki akses untuk melihat list admin yang ada. Pada Datagrid ditampilkan username dan password yang sudah terenkripsi.

Isikan Username dan password dari admin baru. Kemudian tekan tombol Add untuk mengkonfirmasi penambahan permintaan layanan baru. Tunggu hingga proses selesai.

Setelah proses selesai dilakukan, DataGrid kembali melakukan fetch untuk memperbarui data admin
